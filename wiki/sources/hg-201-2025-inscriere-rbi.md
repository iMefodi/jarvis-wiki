# HG 201/2025 — Regulamentul privind modul de efectuare a înscrierii în RBI

**Tip:** hotărâre de guvern + regulament
**Emitent:** Guvernul RM
**Data:** 16.04.2025
**În vigoare:** 16.05.2025 (MO Nr. 233-236 art. 277)
**Modificat:** HG 43/2026 (în vigoare 12.03.2026)
**Temeiul legal:** art. 35 alin. (5) din Legea cadastrului bunurilor imobile nr. 1543/1998
**Fișiere:** [[../../raw/sources/hg-201-2025-inscriere-rbi.pdf]], [[../../raw/sources/HOTARARE Nr. 201/]]

Înlocuiește: Instrucțiunea ARFC 112/2005 privind modul de efectuare a înregistrărilor în RBI.

---

## Obiectele înregistrării (art. 3–4)

**Se înregistrează:**
- Terenurile
- Clădirile și alte construcții principale cu caracter definitiv (clădiri accesorii — la solicitare)
- Încăperile izolate/unitățile în condominiu (apartamente, spații nelocative, locuri de parcare) + cota-parte din dreptul de proprietate/superficie și din părțile comune
- Porțiunile de subsol (perimetrul minier)
- Plantațiile perene (prinse de rădăcini, înființate conform normelor)

**NU se înregistrează:**
- Chioșcuri, gherete, construcții provizorii
- Împrejmuiri, pereți, borne de hotar, strat asfaltic
- Macarale, construcții/instalații funcțional legate de teren/clădire principală și care nu pot fi formate ca obiecte independente (cf. LP 354/2004)
- Obiectele de infrastructură tehnico-edilitară

## Structura Registrului (art. 7–9)

| Capitol | Conținut |
|---|---|
| **A + B** (împreună) | Terenul + construcțiile situate pe acesta |
| **C** | Încăperile izolate (deschis separat pentru fiecare unitate/condominiu) |

Fiecare capitol = 3 subcapitole:

| Subcapitol | Conținut |
|---|---|
| **I** | Date despre bunul imobil (nr. cadastral, adresă, destinație, mod de folosință, suprafață, tipul hotarelor, domeniu) |
| **II** | Dreptul de proprietate (cota-parte, proprietarul, IDNP/IDNO, temeiul înscrierii, data) |
| **III** | Alte drepturi reale și grevări: partea I — superficie, uzufruct, uz, abitație, servitute, ipotecă, drept de folosință UAT, concesiune, drept de administrare; partea II — drepturi de creanță, fapte/raporturi juridice |

**Mijloace de gestiune (art. 10):** plan cadastral + plan geometric, dosar cadastral, dosar tehnic, condica de cereri.

## Procedura de înregistrare (cap. III)

### Cine poate depune cererea (art. 20)
- Persoana care a dobândit dreptul (prin lege, act juridic, moștenire, accesiune, hotărâre judecătorească, act administrativ)
- Oricare dintre părți, dacă actul este autentificat notarial sau emis de autoritate publică
- Reprezentanți legali (pentru minori, persoane protejate)
- Notarul care a autentificat actul — **OBLIGATORIU** prin PDSE (art. 35)

### Termenul de examinare (art. 40)
- **≤ 10 zile lucrătoare** de la data înregistrării cererii
- Prelungire cu **max 40 de zile** dacă sunt necesare documente suplimentare (registratorul emite decizie argumentată)
- Dacă solicitantul nu îndeplinește cerințele în termen → registratorul poate emite decizie de refuz

### Înregistrarea primară — condiție obligatorie (art. 25)
- Există **planul geometric** al bunului imobil, recepționat conform procedurii stabilite

### Depunerea prin notar (PDSE) (art. 35–39)
- Notarul ESTE OBLIGAT să depună cererea electronic prin PDSE după autentificarea contractului translativ sau de ipotecă
- Condiție: imobilul trebuie să fi fost înregistrat anterior de stat; să nu existe interdicții/sechestre care împiedică înregistrarea
- Excepție: notarul poate depune cerere și dacă există sechestre/interdicii, dacă actul prevede dobândirea sub condiție suspensivă a ridicării lor

## Temeiurile de respingere a cererii (art. 30)

| Nr. | Temei |
|---|---|
| 30.1 | Lipsește actul de identitate |
| 30.2 | Persoana nu este autorizată |
| 30.3 | Lipsesc documentele necesare |
| 30.4 | Documentele nu corespund cerințelor legale |
| 30.5 | Dreptul nu este susceptibil de înscriere în Registru |
| 30.6 | Dreptul este deja înregistrat pe numele altei persoane |
| 30.7 | Date din documente ≠ date din cadastru (excepție: eroare OCT) |
| 30.8 | Sechestru sau interdicție notată (excepție: condiție suspensivă de ridicare) |
| 30.9 | Plata nu a fost achitată |
| 30.10 | Înregistrarea trebuie efectuată de alt OCT |
| 30.11 | Documente cu ștersături/adăugiri nemenționate sau executate în creion |
| 30.12 | Documente deteriorate grav |
| 30.13 | File nenumerotate/necoasete (pentru documente cu 2+ file) |
| 30.14 | Bunul sau dreptul nu este individualizat |

## Înregistrarea provizorie (art. 71)

Se efectuează în cazuri speciale, inclusiv:
- **71.1** — dreptul e afectat de condiție suspensivă sau rezolutorie (trebuie indicat expres în actul juridic)
- **71.2** — sechestru/interdicție notată, dar actul prevede dobândirea sub condiție suspensivă de ridicare
- **71.5** — înregistrarea primară a dreptului de proprietate asupra **terenurilor private + case individuale** cu **schema de amplasare** (cf. art. 56¹ Legea 1543/1998)

**Justificarea** înregistrării provizorii (art. 75–77): se efectuează la îndeplinirea condiției → dreptul se consideră înregistrat cu data înregistrării provizorii.

## Nomenclatorul bunurilor imobile (Anexa 4)

### Categorii de terenuri (7 categorii principale)
1. Terenuri cu destinație agricolă (14 moduri de folosință)
2. Terenuri destinate fondului forestier
3. Terenuri destinate fondului apelor
4. Terenuri destinate construcțiilor și amenajărilor (7 moduri)
5. Terenuri destinate transporturilor, comunicațiilor electronice și exploatării miniere
6. Terenuri cu destinație specială (securitate/apărare)
7. Terenuri destinate ocrotirii naturii și patrimoniului cultural

### Clasificarea clădirilor (16 tipuri)
| Cod | Tip clădire |
|---|---|
| 1 | Casă de locuit individuală |
| 2 | Bloc locativ |
| 3 | Clădire publică și administrativă |
| 4 | Clădire comercială, prestări servicii |
| 5 | Clădire de ședere temporară (hotel, pensiune) |
| 6 | Clădire a infrastructurii agricole |
| 7 | Clădire industrială |
| 8 | Clădire de cult |
| 9 | Anexe gospodărești |
| 10 | Garaj (cooperative garaje, individuale) |
| 11 | Clădire nefinalizată (% executat) |
| 12 | Bun viitor (proiectat + autorizat) |
| 13 | Clădire cu statut special (MApN, MAI, SIS etc.) |
| 14 | Clădire cu funcții comasate (mixte) |
| 15 | Clădire ale gospodăriei comunale |
| 16 | Clădire auxiliară (neprincipală, complement la clădire principală nelocativă) |

### Destinația încăperii izolate
- **1 — Locativă** (apartament, cameră în cămin)
- **2 — Nelocativă** (ieșire separată, diverse destinații)
