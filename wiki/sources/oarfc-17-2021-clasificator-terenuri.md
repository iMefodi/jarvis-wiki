# OARFC 17/2021 — Clasificatorul terenurilor după categoria de destinație și folosință

**Tip:** ordin tehnic (clasificator)
**Emitent:** ARFC (acte în vigoare sub AGCC)
**Data:** 19.05.2021 (MO Nr. 129-136 art. 527)
**Modificat:** OARFC28 din 21.04.2022, în vigoare 06.06.2022
**Fișiere:** [[../../raw/sources/OARFC17 2021/clasifcatorul terenurilor_17-md.doc]]
**Sursă PDF:** [[../../raw/sources/oarfc-17-2021-clasificator-terenuri.pdf]]

## Domeniu

Clasificatorul stabilește categoriile de destinație și modul de folosință al terenurilor, utilizat în toate documentele cadastrale pentru completarea informației atributive a obiectului informațional (câmpul „destinație" și „mod de folosință" în RBI).

## Structura clasificatorului

### Categoria 1 — Teren agricol

| Cod | Folosință | Note |
|---|---|---|
| 1.1 | Arabil | Culturi anuale sau ierburi perene |
| 1.2 | Vie | Plantații cu viță de vie |
| 1.3 | Livadă | Pomi fructiferi (măr, păr, dud, persic, prune etc.) |
| 1.4 | Sere, solarii și răsadnițe | Acoperiș din sticlă/plastic |
| 1.5 | Arbuști fructiferi | Coacăză, agriș, zmeură, mure |
| 1.6 | Plantații nucifere | Nuc, migdal, alun, castan, pecan |
| 1.7 | Plante efiro-uleioase | Lavandă, trandafir, șalfei |
| 1.8 | Plante energetice lemnoase | Plop, salcie, salcâm, paulownia, miscanthus |
| 1.9 | Grădină | Loturi de pe lângă casă din extravilan |
| 1.10 | Lot pomicol | Lot din cadrul întovărășirii pomicole |
| 1.11 | Drumuri care servesc activitățile agricole | Deplasare mașini agricole |
| 1.12 | Infrastructura agriculturii | Depozite, frigidere, hale procesare, ferme zootehnice |
| 1.13 | Fâșii forestiere | Fâșii de protecție a câmpului, antierozionale, de protecție drumuri agricole |
| 1.14 | Terenuri degradate | Râpi, alunecări, bolovănișuri, stânci, mocirle |
| 1.15 | Fânețe | Producție de iarbă și plante furajere |
| 1.16 | Pășuni | Pășunatul animalelor |
| 1.17 | Aferent pensiunii agroturistice | Structuri de primire turistică la sat (3–20 camere) |
| 1.18 | Aferent obiectivelor hidroameliorative | Bazine, stații pompare, canale (în circuit agricol) |
| 1.19 | Aferent sistemelor fotovoltaice solare | Terenuri cu sisteme fotovoltaice |

### Categoria 2 — Terenuri din intravilanul localităților

| Cod | Folosință | Note |
|---|---|---|
| 2.1 | Aferent drumurilor publice din intravilan (zona străzilor) | Parte carosabilă, trotuare, piste ciclism, parcaje, acostamente |
| 2.2 | Aferent obiectivelor gospodăriei comunale | Cimitire, stații apă, canalizare, depozite deșeuri, termoficare |
| 2.3 | Aferent obiectivului comercial și prestări servicii | Piețe, centre comerciale, magazine, bănci, autoservice, parcări |
| 2.4 | Aferent obiectivului de cult | Biserici, mânăstiri, case de rugăciuni |
| 2.5 | Aferent obiectivului destinat locuinței | Case individuale, blocuri, cămine, hoteluri, azile |
| 2.6 | Aferent obiectivului public și administrativ | Instituții publice, școli, spitale, stadioane, muzee |
| 2.7 | Pentru amplasarea garajelor | Cooperative garaje, garaje individuale |
| 2.8 | Pentru amenajări și spații verzi | Parcuri, grădini publice, scuaruri, plaje, spații verzi |
| 2.9 | Lot de lângă casă | Proprietate privată intravilan, atribuit în temeiul art.82 Cod Funciar (redacția 1991) |

### Categoria 3 — Terenuri destinate industriei, transporturilor, telecomunicațiilor și cu alte destinații speciale

| Cod | Folosință |
|---|---|
| 3.1 | Aferent drumurilor publice din extravilan |
| 3.2 | Aferent obiectivelor transportului feroviar |
| 3.3 | Aferent obiectivelor transportului aerian |
| 3.4 | Aferent obiectivelor transportului naval |
| 3.5 | Aferent obiectivelor transportului prin conducte |
| 3.6 | Aferent obiectivelor rețelelor electrice |
| 3.7 | Aferent obiectivelor comunicațiilor electronice |
| 3.8 | Aferent obiectivelor cu destinație specială (militar, penitenciar, vamal, geodezic) |
| 3.9 | Pentru amplasarea și exploatarea obiectivelor industriale |
| 3.10 | Pentru exploatări miniere (mine, cariere) |
| 3.11 | Aferent serviciilor rutiere (stații carburant, service, alimentație publică etc.) |

### Categoria 4 — Terenuri destinate ocrotirii naturii, sănătății, activității recreative, istorico-culturale și zone verzi

| Cod | Folosință |
|---|---|
| 4.1 | Aferent obiectivelor de ocrotire a naturii (rezervații, parcuri naționale, grădini botanice) |
| 4.2 | Aferent obiectivelor de tratament natural (izvoare minerale, nămoluri, stațiuni balneare) |
| 4.3 | Aferent obiectivelor pentru odihnă și turism (case odihnă, sanatorii, campinguri, tabere) |
| 4.4 | Aferent obiectivelor istorico-culturale (monumente arheologice, parcuri memoriale) |
| 4.5 | Aferent zonelor suburbane și zonelor verzi (din extravilan — cimitire, stații apă, depozite deșeuri etc.) |

### Categoria 5 — Terenuri ale fondului silvic

| Cod | Folosință |
|---|---|
| 5.1 | Acoperite cu păduri și destinate împăduririi |
| 5.2 | Pentru creșterea materialului forestier de reproducere (pepiniere) |
| 5.3 | Pentru amplasarea și exploatarea obiectivelor de gospodărire forestieră |

### Categoria 6 — Terenuri ale fondului apelor

| Cod | Folosință |
|---|---|
| 6.1 | Aferent iazului (cuvetă + fâșie riverană + construcție hidrotehnic) |
| 6.2 | Aferent lacului |
| 6.3 | Aferent pârâului |
| 6.4 | Aferent râului |
| 6.5 | Aferent mlaștinii |
| 6.6 | Aferent amenajării piscicole (heleșteie, iazuri reproducere pești) |
| 6.7 | Aferent obiectivelor hidro-ameliorative (excluse din circuitul agricol) |

## Utilizare practică

- Codurile se înscriu în câmpul atributiv al obiectului informațional la întocmirea planului geometric și a raportului de lucrare
- Clasificatorul se corelează cu **CF 22/2024** (noul Cod Funciar, în vigoare 07.03.2025) — tabelul de corespondență nu este inclus în clasificator dar este menționat în `cadastru-bunuri-imobile.md`
- **Ordin ARFC 17/2021** modificat prin OARFC28/2022 — anexa actualizată
