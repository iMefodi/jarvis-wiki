# IP CBI — Instituția Publică „Cadastrul Bunurilor Imobile"

**Tip:** instituție publică
**Denumire completă:** IP Cadastrul Bunurilor Imobile
**Site:** ipcbi.gov.md
**Creată:** 03.01.2024

## Descriere

IP Cadastrul Bunurilor Imobile (IP CBI) este instituția publică creată la 03.01.2024 în urma reorganizării ARFC, care preia toate serviciile cadastrale operative. Funcționează sub supravegherea [[agcc]].

## Funcții principale

- Ținerea Registrului Bunurilor Imobile (RBI)
- Prestarea serviciilor de înregistrare a drepturilor imobiliare
- Prestarea serviciilor de informare din RBI
- Recepția și verificarea documentației cadastrale depuse de inginerii cadastrali

## Acte normative principale

- **HG 201 / 16.04.2025** — Regulamentul privind înscrierea în RBI (înlocuiește Instrucțiunea ARFC 112/2005; în vigoare din 16.05.2025)
- **HG 130 / 13.03.2025** — Nomenclatorul și tarifele serviciilor IP CBI (aplicabil din 28.05.2025)

## Context

Înainte de 03.01.2024, serviciile cadastrale erau prestate de [[arfc]] prin oficiile cadastrale teritoriale (OCT). Reorganizarea din 2023–2024 a separat funcția de reglementare ([[agcc]]) de funcția operativă (IP CBI).
